Placeholder Images README
========================

This website uses placeholder images for product and gallery displays.

- Format: JPEG
- Size: 800x600 pixels
- Theme: Neutral stone/texture image that represents granite

The following components reference this placeholder image:
1. Home.js - Product highlights section
2. Products.js - Product listings
3. Gallery.js - Project gallery

For a production website, replace this placeholder with actual product images.
